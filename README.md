# Excel Dashboard Capstone Project

## Overview
This project focuses on data handling, cleaning, analysis, and visualization by analyzing sales, supplier performance, and delivery data using Microsoft Excel.

## Features
- Interactive Dashboard
- Pivot Tables & Charts
- Slicers for filtering
- Analysis of delayed deliveries
- Top 5 suppliers by total cost

## Tools Used
- Microsoft Excel
- Pivot Tables
- Excel Functions (IF, VLOOKUP, COUNTIF)

## Insights
- Identified key suppliers contributing to high costs
- Highlighted delivery delays and operational inefficiencies
- Improved decision-making using visual reports
